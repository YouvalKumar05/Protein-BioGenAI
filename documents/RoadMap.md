# Product Roadmap – TestPilot AutoQA

## Overview
This roadmap outlines the planned releases, key features, dependencies, and demo readiness of the AutoQA platform.

---

## Roadmap Table

| Release | Date | Key Features | Dependencies | Demo Status |
|--------|------|-------------|--------------|-------------|
| v0.1 | 01-Apr-2025 | Authentication (Login/Signup), Basic Dashboard (Overview Metrics), Initial UI Setup | None | Completed |
| v0.2 | 08-Apr-2025 | Data Input Module, AI Scenario Generation, Analysis Review Page | AI Integration (LLM), File Upload Handling | Completed |
| v0.3 | 15-Apr-2025 | Playwright Test Execution, Execution Console (Live Logs Streaming), Basic Reports | Async Backend + Playwright Setup | On Track |
| v0.4 | 22-Apr-2025 | Bug Auto-Creation, Execution Dashboard, Initial Task Management | Execution → Bug Mapping Logic | Planned |
| v0.5 | 29-Apr-2025 | Workflow Automation Engine (Basic), Task Routing, Notifications System | Workflow Engine + Event System | Planned |
| v1.0 | 01-May-2025 | RBAC System (Role Hierarchy, Permissions), Admin Panel, Kanban Board | RBAC Backend + UI Integration | Yet to Plan |
| v1.1 | 08-May-2025 | Communication System (Inbox), AI Insights Dashboard, Advanced Reports | Messaging System + AI Recommendation Engine | Yet to Plan |

---

## Notes

### Release Strategy
- v0.x → Core system (AI + Execution + STLC flow)
- v1.x → Enterprise features (RBAC, Workflow, Communication, AI insights)

### Key Dependencies
- AI Engine → Required for test generation & insights  
- Playwright → Required for automation execution  
- Workflow Engine → Required for task automation  
- RBAC → Required for enterprise access control  

---

## Summary
This roadmap shows a structured transition from core AI-based testing features to advanced enterprise-level workflow automation, role management, and intelligent insights.
